Editores de texto que podemos utilizar para construir os exercícios

Atom.io

O Atom é um editor de texto moderno e acessível. Ele é uma ferramenta que você pode personalizar para fazer qualquer coisa, mas também usar de forma produtiva sem tocar em um arquivo de configuração.

Visual Studio Code

O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft para Windows, Linux e macOS. Ele inclui suporte para depuração, controle Git incorporado, realce de sintaxe, complementação inteligente de código, snippets e refatoração de código.
